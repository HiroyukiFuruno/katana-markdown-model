# katana-markdown-model v0.1.0

## Highlights

- Added a renderer-neutral Markdown document model for the KatanA ecosystem.
- Fixed source ranges, line-column positions, raw snippets, and fingerprints as KMM public DTOs.
- Established `Resolved`, `Moved`, `Unresolved`, and `Conflict` as the public contract for external metadata target resolution.
- Fixed current KatanA fixtures, README badges, alerts, and description lists as canonical fixtures.
- Added the `just harness-up` auxiliary structure-inspection environment.
- Fixed the downstream handoff contract and documented handoff conditions for KDV, KLE, KCF, and KatanA.
