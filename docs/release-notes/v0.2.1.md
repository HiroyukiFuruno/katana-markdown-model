# katana-markdown-model v0.2.1

## Highlights

- GitHub alert blockquote 内の日本語 list item で parser panic が起きないようにした。
- nested inline span を `KmmNode.children` で取得できるようにした。
- one-line dollar math block を `DollarMathBlock` として取得できるようにした。
- KatanA の日本語 sample fixture をリポジトリ内へコピーし、KDV HTML 書き出し（HTML export）向け AST 回帰テストに追加した。

## Boundary

- KMM は HTML / PDF / PNG / JPG を生成しない。
- KMM は viewer / editor の scroll、selection、highlight を制御しない。
- KMM は KDV、KatanA、KCF、KLE、UI framework へ依存しない。
- third-party parser AST は公開契約に含めない。
