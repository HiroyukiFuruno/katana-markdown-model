# katana-markdown-model v0.2.0

## Highlights

- KDV の HTML 書き出し（HTML export）で必要な inline Markdown 構造を、公開データ型（public DTO）として追加した。
- strong / emphasis / strikethrough / inline code / inline HTML / text を `KmmNode.children` から取得できるようにした。
- link / autolink / image / footnote reference / footnote definition / inline math / dollar math block を追加した。
- list item の marker、task marker、本文、nested list、list 内 code block を `ListNode.items` から取得できるようにした。
- blockquote の paragraph / list / code block children を返すようにした。

## Boundary

- KMM は HTML / PDF / PNG / JPG を生成しない。
- KMM は viewer / editor の scroll、selection、highlight を制御しない。
- KMM は KDV、KatanA、KCF、KLE、UI framework へ依存しない。
- third-party parser AST は公開契約に含めない。
